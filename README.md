## 🏆 Projets Réalisés

### 1. Chatbot des Journaux Officiels de Mauritanie
**Technologies :** Python, Streamlit, Ollama, LangChain  
Développement d'un chatbot intelligent utilisant le NLP pour faciliter l'accès et la recherche dans les journaux officiels de Mauritanie.

**Fonctionnalités clés :**
- Recherche intelligente dans les journaux officiels
- Interface conversationnelle naturelle
- Extraction automatique d'informations pertinentes
- Indexation et recherche sémantique

### 2. Application Web d'Analyse de l'INPC
**Technologies :** Python, Django, MySQL, Docker  
Application web pour le calcul de l'Indice National des Prix à la Consommation (INPC) en Mauritanie.

**Fonctionnalités :**
- Collecte et importation des données
- Visualisation dynamique des données
- Calcul automatique de l'INPC

### 3. Projet MEF (Ministère de l'Économie et des Finances)
**Technologies :** Python, Django, MySQL  
Système d'information pour la gestion des données économiques et financières.
